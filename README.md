# chwk
Chatwork CLI tool

## Installation
```
cd ~
wget https://github.com/wintia-yk/chwk/releases/download/v1.0.0/chwk
mv chwk /usr/local/bin/
```

## Usage
```
chkw -t [Chatwork API token] -r [Chatwork room id] -m [Sending message]
```
